## 基于Laravel框架搭建的律所管理系统（开发中）
<p align="center"><img src="http://supercat.cc/show/1.png"></p>

项目名称律所管理系统，目前功能：

前台：
- 律师专栏：律师账号相关操作
- 个案查询：用户查询个人案件、案件材料
- 常法服务中心：常法客户会务记录、合同修改、微信沟通群
- 微信沟通群：查询用户用户专属沟通群
- 关于我们：团队展示
- 律师搜索

后台：
- 客户管理
- 律师管理
- 行政管理
- 案件管理
- 常法合同管理
- 文章管理
- 权限管理
